# Shortlet Business Intelligence Dashboard

A Power BI dashboard designed to monitor the performance of a shortlet apartment business across revenue, occupancy, guest behavior, and operational efficiency.

The solution provides executive-level insights into revenue performance, booking channels, guest segments, and property operations.

## Business Problem

Shortlet operators need insights into:

- Revenue performance
- Occupancy trends
- Booking channel effectiveness
- Guest behavior patterns
- Operational efficiency

This dashboard consolidates these metrics into a multi-page Business Intelligence solution for decision-making.

## Dashboard Pages

### Executive Overview
- Gross Revenue
- Net Revenue
- Occupancy Rate
- ADR
- RevPAR
- Revenue Trends
- Revenue Contribution by Location

### Revenue & Booking
- Booking Trends
- Revenue by Channel
- RevPAR Analysis
- ADR Analysis
- Revenue Movement

### Operations & Guests
- Guest Type Analysis
- Guest Booking Behaviour
- Cancellation Analysis
- Operational Metrics

## Key Insights

- Generated ₦46M Gross Revenue and ₦33.4M Net Revenue
- Achieved 64.2% Occupancy Rate
- Victoria Island contributed 46.2% of total revenue
- Airbnb remained the strongest acquisition channel
- Business guests generated the highest revenue contribution

## What I Learned

This project reinforced that effective BI is more than visualization.

Key lessons included:

- Structuring dashboards around business questions
- Separating executive, operational, and analytical views
- Using KPI variance analysis instead of standalone metrics
- Applying UX principles to dashboard navigation and information hierarchy
- Building dashboards as decision-support tools rather than reporting tools

## How to Use
1. Download the `.pbix` file
2. Open with **Power BI Desktop**
3. Use the slicers (Month, Location, Channel) to explore
